#使用说明：
###step1:把仓库clone下来
##### git clone "https://github.com/zhanyuzhang/ES6-and-React.git"
###step2: 安装依赖
#####打开终端，进入到ES6-and-React目录，然后输入： npm  install，然后等待数分钟
###step3: 启动app
#####安装好依赖之后，终端输入：npm start
###step4: 预览
#####默认端口为8080，所以，浏览器地址栏输入：http://localhost:8080/即可
###注意：
#####确保电脑已经安装了npm和node
